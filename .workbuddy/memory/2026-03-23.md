### 重大功能更新 (17:10) 🎉
**用户反馈**: K线显示需要改进为蜡烛图形式，要求显示准确的OHLC，新增中国现货黄金切换

**实现内容**:
1. **真正的蜡烛图绘制**：
   - 使用Canvas自定义绘制，而非Chart.js
   - 画出真实的蜡烛形状：影线（Wicks）+ 蜡烛体（Body）
   - 影线：显示High-Low范围
   - 蜡烛体：显示Open-Close范围
   - 红涨绿跌配色
   - 自动Y轴缩放
   - 网格线和刻度

2. **新增价格线图**：
   - 显示收盘价连续走势
   - 蓝色填充线图表
   - 便于观察整体趋势

3. **黄金价格类型切换**：
   - 国际黄金（USD/oz）
   - 中国现货黄金（¥/克）
   - 自动调整数据范围和单位标签

4. **交互改进**：
   - 鼠标悬停显示详细OHLC信息
   - 窗口改变大小自动重绘Canvas
   - 响应式设计优化

**技术细节**:
```javascript
// 蜡烛图绘制核心
1. 获取canvas和2D绘图上下文
2. 设置画布宽高 = 容器宽高
3. 计算价格范围（min/max）+ 10%边距
4. 将价格转换为Y坐标
5. 遍历数据绘制蜡烛：
   - 影线：x中心点, high->low
   - 蜡烛体：矩形, open->close
6. 添加鼠标交互
```
