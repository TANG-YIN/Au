### 真实API集成与交易时间管理 (17:18) 🚀
**用户反馈**: 数据不对，收盘后不应该有波动

**问题分析**:
1. 之前的数据完全是随机模拟的
2. 非交易时段（21:00-09:00）应该没有价格波动
3. 需要集成真实黄金价格API

**解决方案**:
1. **集成真实API**：
   - 国际黄金：free.xwteam.cn/api/gold/trade (伦敦金/港金)
   - 中国现货：v2.xxapi.cn/api/goldprice (银行金条价格)
   - 使用真实基础价格生成历史数据

2. **交易时间管理**：
   ```javascript
   const hour = date.getHours();  // 0-23
   const isMarketOpen = hour >= 9 && hour < 21;  // 09:00-21:00
   
   if (isMarketOpen) {
       // 交易时间：正常波动
       volatility = randomWalk + trend;
   } else {
       // 非交易时间：无波动（保持前一个价格）
       volatility = 0;
   }
   ```

3. **数据结构改进**：
   - 每个蜡烛数据添加 `isMarketOpen` 字段
   - 鼠标悬停显示交易时间标记
   - 非交易时段的K线体积会很小

4. **时间框架调整**：
   - 1天（1小时粒度）
   - 7天（1小时粒度）
   - 30天（1小时粒度）

**技术实现**:
- API调用使用 Fetch API
- 错误处理：API失败时使用默认价格
- 历史数据：基于真实API价格生成
