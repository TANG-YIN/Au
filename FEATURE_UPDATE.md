## 使用指南

### K线图（蜡烛图）
1. 选择黄金价格类型（国际/中国）
2. 选择时间框架（6h/1d/3d/7d）
3. 鼠标悬停查看详细信息

**蜡烛解读**：
- 红色蜡烛：价格上涨（收盘 > 开盘）
- 绿色蜡烛：价格下跌（收盘 < 开盘）
- 蜡烛高度：开盘到收盘价差
- 影线长度：当日最高最低价差

### 价格线图
- 直观显示价格趋势
- 便于观察大方向

### 分析指标
- **MA5/MA10/MA20**：短期平均走势
- **RSI**：强弱指标，判断超买超卖

## 技术实现

### 蜡烛图绘制
```javascript
// 核心逻辑：
1. 遍历数据中每条蜡烛
2. 计算影线：high -> low 的垂直线
3. 计算蜡烛体：max(open,close) -> min(open,close) 的矩形
4. 上涨（红色）vs 下跌（绿色）
5. 添加鼠标交互提示
```

### Canvas缩放处理
```javascript
// 自动适应容器大小
canvas.width = container.clientWidth;
canvas.height = container.clientHeight;

// 窗口改变大小时重绘
window.addEventListener('resize', () => {
    createCandlestickChart(currentData);
});
```
