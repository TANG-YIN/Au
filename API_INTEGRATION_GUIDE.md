## 更新时间

## 核心改进

### 1. ✨ 集成真实黄金价格API

#### 国际黄金数据源
**API**: `https://free.xwteam.cn/api/gold/trade`
- 返回伦敦金、港金等国际行情
- 单位：USD/oz（美元/盎司）
- 包含实时价格、当日高低价

#### 中国现货黄金数据源
**API**: `https://v2.xxapi.cn/api/goldprice`
- 返回银行投资金条、品牌首饰价格
- 单位：¥/克（人民币/克）
- 包含多家银行金条价格

**数据示例**：
```json
// 国际黄金
{
  "code": "LHJD",      // 伦敦金代码
  "price": 2085.5,     // 当前价格
  "updated": "2026-03-23 17:18"
}

// 中国现货
{
  "bank": "工商银行",
  "price": "435.50",   // ¥/克
  "updated_date": "2026-03-23"
}
```

### 2. 🕐 交易时间管理（重要）

**核心改进**：收盘后不再有价格波动

**交易时间定义**：
- **交易时段**：09:00 - 21:00（每小时有正常波动）
- **非交易时段**：21:00 - 09:00（保持上一个收盘价，无波动）

**实现逻辑**：
```javascript
// 获取当前小时
const hour = date.getHours();

// 判断是否交易时间
const isMarketOpen = hour >= 9 && hour < 21;

if (isMarketOpen) {
    // 交易时间：正常波动
    volatility = randomWalk + trend;
} else {
    // 非交易时间：无波动（保持前一个价格）
    volatility = 0;
}
```

**K线图显示**：
- 鼠标悬停时显示 `(交易时段)` 或 `(非交易时段)` 标记
- 非交易时段的蜡烛体会很小或不存在（开盘=收盘）

### 3. 📊 数据精确度提升

**历史数据生成**：
- 基础价格：从真实API获取
- 历史数据：基于基础价格生成合理波动
- 日期范围：精确到小时级别
- 价格精度：2位小数

### 4. 🎯 时间框架优化

| 框架 | 数据点 | 间隔 | 时间跨度 |
|------|-------|------|--------|
| 1天 | 24 | 1小时 | 24小时 |
| 7天 | 168 | 1小时 | 7天 |

**说明**：

## 工作原理

### 初始化流程
```
1. 页面加载
   ↓
2. 调用 fetchRealGoldData('international')
   ↓
3. 从 free.xwteam.cn API 获取实时价格
   ↓
4. 使用获取的价格作为基础价格
   ↓
5. 调用 generateHistoricalData() 生成历史数据
   ↓
6. 根据交易时间管理规则生成蜡烛图
   ↓
7. 绘制所有图表
```

### 数据流转
```
API数据 → 基础价格 → 历史数据生成 → 交易时间管理 → 蜡烛图渲染
                  ↓
              技术指标计算 (MA5/MA10/MA20/RSI)
                  ↓
              多图表展示
```

### 交易时间判断
```javascript
const hour = date.getHours();  // 0-23
if (hour >= 9 && hour < 21) {
    // 09:00-21:00 为交易时间
} else {
    // 其他时间无波动
}
```

如果API请求失败，系统会：

3. 显示信息提示用户
