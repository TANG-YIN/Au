## 🎮 如何使用

### 访问网站
直接点击或复制链接到浏览器：
```
https://tang-yin.github.io/Au/
```

### 切换K线时间框架
点击顶部按钮：
- **6小时** - 查看短期走势
- **1天** - 查看日线走势
- **3天** - 查看中期走势
- **7天** - 查看周线走势

### 显示/隐藏指标
点击折线图下方的指标按钮：
- **MA5** / **MA10** / **MA20** - 移动平均线
- **RSI** - 相对强弱指标

### 查看价格详情
- 将鼠标移至K线上 → 显示OHLC数据
- 查看底部 → 实时显示当前价格
