# GitHub Pages 部署说明

## 📦 项目已成功推送到 GitHub

你的黄金价格可视化项目已经成功部署到 GitHub 仓库：https://github.com/TANG-YIN/Au

## 🌐 启用 GitHub Pages 的步骤

由于 GitHub CLI 在当前环境中不可用，请按照以下步骤通过网页界面启用 GitHub Pages：

### 步骤 1：打开仓库设置
1. 访问：https://github.com/TANG-YIN/Au
2. 点击仓库顶部的 **Settings** (设置) 标签

### 步骤 2：配置 Pages
1. 在左侧菜单中找到 **Pages** 选项并点击
2. 在 **Source** 部分，选择 **Deploy from a branch**
3. 分支选择：**main**
4. 文件夹选择：**/ (root)**
5. 点击 **Save** (保存) 按钮

### 步骤 3：等待部署
- 通常需要 **1-3 分钟** 完成首次部署
- 部署状态会在页面顶部显示
- 完成后会显示绿色的✓标记

## 🎉 部署完成后的访问 URL

启用 GitHub Pages 后，你的网站将可以通过以下 URL 访问：

```
https://tang-yin.github.io/Au/
```

## ⚠️ 重要提示

- 如果 `tang-yin.github.io` 不是你的 GitHub 用户名，请替换为你的实际 GitHub 用户名
- 例如：如果你的 GitHub 用户名是 `zhangsan`，URL 将是 `https://zhangsan.github.io/Au/`
- 部署完成后，你也可以在 GitHub Pages 设置页面看到确切的 URL
